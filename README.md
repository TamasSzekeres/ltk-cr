# ltk-cr

Lightweight Windowing Toolkit for Crystal language.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  ltk:
    github: TamasSzekeres/ltk-cr
    branch: master
```

Then run in terminal:
```bash
crystal deps
```

## Usage


```crystal
require "ltk"

module YourModule
  include Ltk
end
```
